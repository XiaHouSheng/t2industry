## 📌 项目定位
一款面向《明日方舟：终末地》的多功能工具平台，核心能力包含：
- 📊 物料配平计算
- ⚡ 生产效率模拟
- 📤 蓝图分享与管理

## 🛠 技术栈
- 前端框架：Vue 3
- UI 组件库：Element Plus（高效搭建交互界面）
- 网格编辑：Gridstack.js（快速实现可视化网格场景编辑）
